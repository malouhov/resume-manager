# Resume Manager

## Introduction
I am excited to introduce this persistence project which is developed to manage job applications.

## Main goals
* create a resume management tool;
* dive deep into Spring persistence technologies and concepts.

## Technologies
* Java 17
* Spring Boot
* Spring Data JPA
* Hibernate
* PostgreSQL

## Description

### Database Tables
* Companies
* Vacancies
* Tags
